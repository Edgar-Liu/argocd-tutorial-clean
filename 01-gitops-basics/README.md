# Module 01: GitOps Basics with ArgoCD

Learn the fundamentals of GitOps by deploying an application with ArgoCD.

## What You'll Learn

- How ArgoCD monitors Git and syncs Kubernetes clusters
- Building and pushing container images
- Deploying with ArgoCD (auto-sync, self-heal)
- Updating deployments via Git (the GitOps way)
- Setting up CI/CD for automated deployments

## Choose Your Guide

- [KIND + Docker Hub](docs/guide-kind.md) — local setup, no cloud needed
- [EKS + ECR](docs/guide-eks.md) — production-like, uses AWS
